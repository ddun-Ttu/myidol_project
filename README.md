## 서비스 소개
---

언제나 아이돌과 함께하는, 팬을 위한 판매 앨범 사이트
아이돌 팬들을 위한 특별한 판매 사이트입니다. **"마이아이돌"**은 사랑하는 아이돌의 음악과 추억을 손쉽게 만날 수 있는 곳입니다.
